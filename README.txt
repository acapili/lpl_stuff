Prerequisite: 
    Download and install Python (32 bit) onto your machine
    Copy hasp_rt.exe (found in 1LPFvv15) to the python folder (1LPFvv15 -> OlxAPI -> python)

This program is currently lacks any sort of GUI but has been programmed to facilitate a GUI if deemed necessary.

Current usability: 
    Display all zones, areas, and relays. 
    Given two buses that are connected it will give the relay coordination between the two. Gives an error if the bus does not exist or the buses are not connected
